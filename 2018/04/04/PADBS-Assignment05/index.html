Problem

Based on database of Student – Course, write the relation algebra expresses. (One express for each question).
Student(sno, sname, ssex, sdept, sage)
Course(cno, cname, cpno, ccredit)
SC(sno, cno, grade)

  1. Query the student name (sname) who selected all courses except course 8 and course 9. (Using division)
  2. Query the course name (cname) and its prerequisite’ prerequisite’ course number (cno).
  3. Attach dbbeers and dbstudents database. Paste all the tuples of table Bars in dbbeers database and all the tuples of table course in dbstudents into this document.

Query the student name (sname) who selected all courses except course 8 and course 9.

Projsname {Student Join [Projsno, cno(SC) ÷ Projcno(Selectcno≠8 and cno≠9(Course)) - Projsno(Selectcno=8 or cno=9(Course))]}

Query the course name (cname) and its prerequisite’ prerequisite’ course number (cno).

Projcname, cppno{Projcno, cname, cpno(Course) Join RenameC(cpno, cppno)[Projcno, cpno(Course)]}

常见问题

  • 第一问类似Projsname {Student Join [Projsno, cno(SC) ÷ Projcno(Selectcno≠8 and cno≠9(Course))]}
    • 这种做法会让最后的结果变成了选除课程8和9的学生+选除课程8的学生+选除课程9的学生。当然这三部分有共同部分选除课程8和9的学生
  • 第一问类似Projsname(Student Join {[Projcno, sno(Selectcno≠8 and cno≠9(SC))]/Selectcno≠8 and cno≠9(Course)})
    • 这种做法和前一种类似，Projcno, sno(Selectcno≠8 and cno≠9(SC))并没有改变除了8和9之外的cno对应的sno，因此最后得到的sno依然包含选除课程8的学生+选除课程9的学生
  • 出现表达如 cno != 8 and 9：
    • Sno只能有一个值，不可能既是8又是9，所以需要表达成 cno≠8 and cno≠9
  • 注意关系代数的优先级，优先级见本答案表格，否则就需要用括号标明
