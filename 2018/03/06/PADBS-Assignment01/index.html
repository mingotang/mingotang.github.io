软件安装部分提示和注意事项

软件安装

任何版本的SQL都可以分为 server 和 client 两部分，课程中使用的MS-SQL在 windows 系统上安装完成之后就已经拥有了这两部分。

如果不想使用 MS-SQL，也可以使用 MySQL 或者 Sqlite 。对于不喜欢安装大型软件的同学而言我推荐 Sqlite，该文件式关系型数据库在终端下使用非常方便。

使用 Mac OS 系统的同学可以使用 MySQL 作为 server，DBeaver 作为 client

Mac系统可以使用DBeaver+MySQL的方式代替，请参考：Mac OS MySql 连接和使用示例
