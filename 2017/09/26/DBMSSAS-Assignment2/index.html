  • TA
Tuesday, September 26th 2017

Problem

Design a Student – Course Database
Draw E/R Diagram

Answer

实体：学生 Student

  • 学号 Sno（关键字，需要加下划线）
  • 姓名 Sname
  • 性别 Ssex
  • 年龄 Sage
  • 学院 Sdept

注意：每个实体（表）都需要一个关键字来进行索引

实体：课程 Course

  • 课程编号 Cno（关键字，需要加下划线）
  • 课程名称 Cname
  • 先修课程 Cpno
  • 学分 Ccredit

关系：选课 SC

  • 分数 Grade

实体对应关系：

  • 实体学生与实体课程形成关系选课

常见问题

  1. 实体的属性当中没有键
    • 键用下划线标出
  2. 关系和实体没有属性
    • 关系和实体必须要有属性，如果不存在属性那么这个关系或者实体也就不需要储存在数据库当中了
